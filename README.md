# Калькулятор расписания
Позволяет составить расписание в зависимости от выбранных параметров.  

---
###Используемые технологии:
 - React
 - MobX
 - CSS Modules, SCSS, ClassNames
 - Дополнительные бибиотеки:
   - react-select
   - flatpickr
   - dayjs
